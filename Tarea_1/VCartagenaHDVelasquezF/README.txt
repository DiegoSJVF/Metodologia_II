La compilacion del programa es la siguiente

gcc -o ejecutable.exe Tarea_1.c
./ejecutable.exe -o n k >> Tiempos_1.txt

La ejecucion del programa es la siguiente

./Suficiencia.exe -1 n k   (primer metodo)
./Suficiencia.exe -2 n k   (segundo metodo)
./Suficiencia.exe -3 n k   (tercer metodo)

tal que n >= k y n, k pertenecen a los enteros positivos
